# EduFlow ERP & CRM System

Enterprise ERP and CRM platform designed for educational centers and language academies.

---

## Core Features

- Student Management
- CRM & Lead Tracking
- Sales Pipeline
- Scheduling System
- Teacher Management
- Attendance Tracking
- Installments & Finance
- Reports & Analytics

---

## Tech Stack

- Laravel
- PHP
- MySQL
- TailwindCSS
- REST APIs
